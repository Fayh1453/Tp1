Creation du fichier facture



Création d'un nouvelle variable String (enregistrement) dans la methode affichage (classe Facture) initialiser a "";


Toujours dans la méthode Affichage, rajout des informations afficher dans la variable enregistrement.

Appelle d'une methode enregistrer avec la variable enregistrement comme attribut


Methode enregistrer(String enregistrement)

La méthode enregistrer creer une nouvelle variable Sring qui a pour nom titre avec pour valeur Facture-du-date-heure.txt

utilise un PrintWriter avec la variable enregistrement pour nom la variable titre.
